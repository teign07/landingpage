# ReEnchanted Landing Page

Static marketing site for ReEnchanted.

## Local preview

Run:

    npm run preview

Then open http://localhost:4173.

## Validation

Run:

    npm run check

The checker verifies that local static references used by the HTML, CSS, and JavaScript resolve to files in this repository.

## Deployment

This repository is set up for GitHub Pages using GitHub Actions. Push to main, then set the repository Pages source to GitHub Actions if it is not already enabled.

The site should publish at: https://teign07.github.io/landingpage/
